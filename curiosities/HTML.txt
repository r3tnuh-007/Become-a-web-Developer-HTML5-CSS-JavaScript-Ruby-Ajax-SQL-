
HTML is a markup language and not
a Programming language

HTML defines the structure of your content 
using markup

HTML consists of a series of elements 

The elements are used to wrap different parts of the
content to make it appear a certain way
or act a certain way

HTML elements are the building blocks of HTML pages 
HTML elements are represented by tags <opening> and </closing>

HTML tags can be used to label pieces of content on a website 
such as heading paragraph etc
